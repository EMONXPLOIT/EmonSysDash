*   **Terminal Auto-Flush:** Automatically purges the screen upon launch to clear history noise and build a secure interface.

*   **IP Intelligence Routing:** Parses geolocation data from encrypted network relays to map live Public IP, accurate Country, City, Region, and local ISP channels.

## 🚀 How to Run (Step-by-Step Installation)

### 📌 Option A: Quick Run in Termux (Android)
Copy and paste these commands into your Termux terminal one by one:
```bash
apt update && apt upgrade -y
pkg install python -y
pip install requests
python emonsysdash.py
📌 Option B: Standard Linux / macOS Deployment
​Open your terminal and execute the following configuration pipeline:
```bash
sudo apt update
sudo apt install python3 python3-pip -y
pip3 install requests
python3 emonsysdash.py
📌 Option C: Windows PowerShell / CMD Setup
​Ensure Python is added to your System PATH, then trigger your console:
pip install requests
python emonsysdash.py
⚙️ Advanced Runtime Argument Flags
​You can bypass standard sequential auditing by triggering isolated automated controllers:
Trigger Local Hardware Inspection Module:

Trigger Remote Network Footprint Scanning Module:
